# 🇮🇳 Bharat Health Card

A citizen-first open-source project to provide every Indian with a secure, portable **Digital Health Identity Card**, accessible across hospitals and clinics nationwide.

---

## 📌 Vision

Build a unified digital health infrastructure where a patient's entire medical history — from prescriptions to reports — is linked to a unique digital health card and accessible securely from anywhere.

---

## 🏗️ Tech Stack

| Layer     | Technology        |
|-----------|-------------------|
| Backend   | Django REST       |
| Frontend  | ReactJS           |
| Database  | PostgreSQL (or SQLite for demo) |
| Auth      | Basic Token Auth (JWT for future) |

---

## 🔧 Key Features (MVP)

- Unique Health ID linked to Aadhaar (optional for now)
- Add & view medical records + prescriptions
- Doctor login & patient selection
- Geo-tagging via Indian states & districts
- Basic responsive frontend (React)

---

## 🇮🇳 India-Centric Design

- Aadhaar & PIN support
- Indian states/districts in patient profile
- MCI/NMC doctor registration number field
- Hindi/English support (planned)
- Indian date/time formats
